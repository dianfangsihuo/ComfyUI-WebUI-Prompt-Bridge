# ComfyUI WebUI Prompt Bridge v0.4.29

这次更新主要修复提示词区域的拖动和布局恢复问题，让 Prompt、标签和 LoRA 面板更容易调整，也更不容易在刷新或重新打开工作流后变形。

## 给用户的更新内容

- **分隔条不会再被遮住**：修复提示词区底部的拖拽边框被标签、LoRA 或滚动面板盖住的问题。现在可以正常拖动调整 Prompt、标签和反向词区域的高度。
- **卡片区新增明显拖拽条**：Prompt All in One 卡片选择区域底部增加了可见的蓝色拖拽条，可以直接调整卡片区高度，双击可恢复默认高度。
- **LoRA 折叠更稳定**：折叠或展开 LoRA 后，不会再因为自动回收空白而改写 Prompt/LoRA 分栏；工作流保存和恢复时会保留原来的分栏高度。
- **恢复工作流更顺手**：重新打开或切换工作流后，提示词区域会自动回到顶部，避免滚动位置让分隔条跑到视野外。

## 兼容说明

- 不改变已有 Prompt 文本、收藏、工作流节点、连线和布局缓存格式。
- 更新后请重启 ComfyUI，并在浏览器中按 `Ctrl+F5` 强制刷新，确保新的前端脚本生效。

## Comfy Registry

本版本将 `pyproject.toml` 版本更新为 `0.4.29`，并保留现有注册表元数据。推送到 `main` 后，GitHub Actions 会自动将新版本同步到 Comfy Registry。
